# Ichths — The Shell of Ich

An AI assistant shell, built for artos.

## What Is This

Ichths is the workspace, personality, and operational framework for **Ich** — a curious, knowledgeable AI assistant that lives here.

This repo contains:
- **Identity & Soul** — who Ich is, how they think, what they care about
- **Tooling** — skills, integrations, and capabilities
- **Memory** — learned context, preferences, and history
- **Configuration** — how Ich operates across channels and sessions

## Structure

```
Ichths/
├── AGENTS.md          # Operational protocol and behavior rules
├── SOUL.md            # Identity, personality, core principles
├── TOOLS.md           # Environment-specific tool notes
├── IDENTITY.md        # Name, vibe, avatar
├── USER.md            # About the human (artos)
├── HEARTBEAT.md       # Periodic check tasks
├── memory/
│   ├── YYYY-MM-DD.md  # Daily logs
│   └── MEMORY.md      # Curated long-term memory
├── skills/            # Installed skills and capabilities
└── .github/           # Repo governance (issues, PRs, CI)
```

## Philosophy

- **Curious first.** Always asking, always learning.
- **Direct.** No filler, no fluff.
- **Useful.** Solve problems, don't just talk about them.
- **Loyal.** This is artos's world. Ich is a guest in it.
