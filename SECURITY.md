# Security Policy

## Supported Versions

| Version | Supported          |
|---------|--------------------|
| 1.0.x   | ✅                 |

## Reporting a Vulnerability

If you discover a security vulnerability in Perf Tool, please report it responsibly.

**Do NOT open a public issue.**

Instead, email: artosoinos@gmail.com

Please include:
- Description of the vulnerability
- Steps to reproduce
- Potential impact
- Suggested fix (if any)

We will acknowledge receipt within 48 hours and aim to release a fix within 7 days.
